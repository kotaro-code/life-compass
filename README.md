# Life Compass

人生設計を、もっと身近に。

Life Compass は、家計簿ではなく「人生全体」をシミュレーションするライフプランアプリです。

## コンセプト

結婚、出産、住宅購入、教育費、老後などのライフイベントをもとに、
将来の資産推移をシミュレーションします。

## 開発環境

- Next.js
- TypeScript
- React
- Supabase
- PostgreSQL
- Prisma
- Vercel

## 開発方針

- AI（ChatGPT・Claude Code）を活用した開発
- GitHubで履歴管理
- 小さく作って少しずつ改善する
